# Frontend — Consultorio Odontológico

Angular 19 SPA con Angular Material. Ver el [README raíz](../README.md) para el contexto completo del proyecto.

## Desarrollo

```bash
npm install
npm start
```

Queda en `http://localhost:4200`. Requiere el backend corriendo en `http://localhost:8080`.

## Comandos útiles

| Comando | Descripción |
|---------|-------------|
| `npm start` | Servidor de desarrollo con live reload |
| `npm run build` | Build de producción en `dist/` |
| `npm test` | Tests unitarios con Karma |
